# pintpoint

### A Simple Personal Beer Tracking App

Track and rate beers you come across.

## Roadmap

1. Add test suite.
2. Add ratings for individual beer products.
3. Allow deletion.
4. Integrate with Auth0.
5. Create backend API for persistence, instead of using the current browser storage.
6. Add image uploads.
7. Improve transitions.
8. Update beer styles list based on BJCP, CAMRA, and the Brewers Association.
9. Save location last purchased.
10. Get suggestions for similar beers.


## WARNING

**THIS IS UNDER ACTIVE DEVELOPMENT. DO NOT USE IT TO TRACK YOUR BEER CONSUMPTION.... YET.**
